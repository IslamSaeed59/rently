

## أولاً: ملخص كود البايثون (`extract_id.py`)
هذا الجزء مسؤول عن نظام التحقق من بطاقة الرقم القومي المصري باستخدام الذكاء البصري:
1.  **OCR (تعرف ضوئي):** يستخدم مكتبة `EasyOCR` لقراءة النصوص العربية والإنجليزية من صور البطاقات.
2.  **استخراج البيانات:** يقوم باستخراج الرقم القومي (14 رقم) بدقة، والتحقق من صحته.
3.  **التحقق الذكي:** يتأكد من أن الصورة المرفوعة هي فعلاً بطاقة رقم قومي من خلال البحث عن كلمات مفتاحية (مثل "بطاقة تحقيق الشخصية").
4.  **التكامل:** يرسل البيانات المعالجة إلى سيرفر Node.js لاتخاذ قرار آلي بتوثيق حساب المستخدم.

---



1. إعداد الباك إند (Backend)
1. افتح المجلد `Backend` في الـ Terminal.

   npm install
2. .env 


### 4. إعداد البايثون (مهم جداً لنظام الهوية)
يجب تثبيت المكتبات اللازمة لمعالجة الصور. افتح الـ Terminal واكتب:
pip install easyocr opencv-python torch torchvision


### 5. إعداد الفرونت إند (Frontend)
1. افتح المجلد `FrontEnd` في الـ Terminal.
2. npm install


### 6. تشغيل المشروع
لتشغيل السيرفر والواجهة الأمامية معاً، اذهب لمجلد `Backend` واكتب:
npm run dev
